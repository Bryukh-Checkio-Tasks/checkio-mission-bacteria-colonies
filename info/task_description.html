Biologists use bacteria to clone DNA for sequencing. These bacteria are injected with the DNA that needs to be cloned and then allowed to reproduce on a large plate. As these bacteria reproduce they form colonies. Only bacteria from healthy colonies will be used for sequencing. And here bioinformatics take the game and use techonologies to automitize these process.

Let's represent a plate snapshot as a pixel grid. You are given this grid as a binary matrix where 1 is bacteria cell and 0 is empty. Our goal is find the largest healthy bacteria colony that grown uniformly. Bacteria can grow in four adjacent cells and bacteria colony is a set of bacterias that are connected. The healthy colonies are looked as it's represented on the image.

On the next image unhealthy colonies are marked by orange colors. And remember - single cell is not a colony, so it can not be counted as healthy colony.

You should find the largest healthy colony and return coordinates of the center cell. The top-left cell has coordinates (0, 0). If on the plate no a healthly colony, then return (0, 0) (or [0, 0]). If on the plate several largest colonies with the same size, then return any of them.

Sum two passed ints

Input: A grid as a tuple of tuples with integers 1/0.

Output: The coordinates of the largest colony as a list or a tuple of two integers.
