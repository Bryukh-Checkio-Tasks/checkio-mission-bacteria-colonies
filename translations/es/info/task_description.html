Los biólogos utilizan bacterias para clonar ADN en “secuenciación”. Estas bacterias se inyectan con el ADN que quiere ser clonado, y se deja reproducir en grandes placas (plates). A medida que dichas bacterias se reproducen, empiezan a formar colonias y solo las bacterias de colonias sanas serán utilizadas para la secuenciación. Es en este punto en donde la bioinformática entra al juego, implementando tecnologías para automatizar estos procesos.

Vamos a representar una imagen instantánea de una placa mediante una cuadrícula de píxeles. Se te entregará la cuadricula como una matriz binaria en donde 1 representa una célula de bacteria y 0 es un espacio vacío. Nuestro objetivo es encontrar la mayor colonia de bacterias saludables que haya crecido de manera uniforme. Las bacterias pueden crecer en cuatro celdas adyacentes y una colonia de bacterias es un set de bacterias conectado. Las colonias sanas tienen un aspecto similar a las siguientes imágenes:

A continuación podemos ver que colonias poco saludables están marcadas con un color anaranjado. Recuerda que una sola célula no es una colonia, por lo que no se puede considerar como una colonia sana.

Tu misión es encontrar la mayor colonia saludable e informar las coordenadas de su celda central. La celda superior izquierda tiene las coordenadas (0, 0). Si no existe una colonia sana en la cuadricula, debes reportar (0, 0) (o [0, 0]). Si existen varias colonias con el mismo tamaño (el mayor) puedes reportar cualquiera de ellas.

Datos de Entrada: La cuadricula como una tupla de tuplas, con números enteros (int) 1/0.

Salida: Las coordenadas del centro de la colonia más grande, como una lista o tupla de dos enteros (int).
