生物学者はシークエンシング(塩基配列の解析)のためにDNAをクローンするとき、バクテリアを使います。 クローンする必要があるDNAを注入されたバクテリアを大きな培地(plate)で繁殖させます。 バクテリアは繁殖するにつれて、コロニーを形成しはじめます。 健康なコロニーのバクテリアだけが、シークエンシングに使われます。 そしてここで生物情報学(bioinformatics)をゲームにして、このプロセスを自動化するためのテクノロジーを使います。

培地のスナップショットをピクセルのグリッドで表現します。 あなたは、このグリッドを1-0行列で与えられます。ここで1はバクテリアのセルを0は空白を表します。 我々の目的は一様に成長した健康なバクテリアコロニーのうちで一番大きいものを発見することです。 バクテリアは4つの隣り合ったセルへ成長することができ、バクテリアコロニーは連結したバクテリアの集団です。 健康なコロニーは下の図で示したような形をしています。

次にオレンジ色にマークした不健康なコロニーを見てみます。 単一のセルはコロニーではないので、健康なコロニーとして数えることができません。

あなたのミッションは最大の健康なコロニーを発見して、中央のセルの座標を返すことです。 左上のセルの座標は(0, 0)です。 培地に健康なセルがないとは、(0, 0)または[0, 0]を返します。 培地に同じ大きさの最大コロニーが複数あったときは、それらを全部返します。

入力: グリッド、整数1/0のtupleのtuple。

出力: 最大コロニーの座標のlist、2つの整数のlistのlist。
